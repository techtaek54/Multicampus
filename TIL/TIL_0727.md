# String
- python은 다른 언어에 비해 str을 자유롭게 관리할 수 있음
- `container` `sequntial` `iteralbe` `immutable`

## Method
- `.split()`
- `.strip('char')` : 양쪽 끝 특정 문자 제거
- `.find()` `.index()`
- `.count()`
- `.replace()`
- `.join(iterable)` : to str

## ASCII code
- 컴퓨터는 이진수 체계
- 따라서 문자를 표현하기 위해 정해진 규약을 따름 => ASCII code
- 관련 함수 : `ord()` `chr()`
